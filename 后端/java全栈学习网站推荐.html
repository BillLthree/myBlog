    Java学习网站推荐😀

    Yuan hui 2021/9/1 Java

    # [Github] java||全栈 学习网站推荐

    # JavaFamily

    https://github.com/AobingJava/JavaFamily

    包括java学习路线、java基础知识 (opens new window)串讲、计算机操作系统、IO、Linux、计网、设计模式、集合框架、并发、消息队列、数据库等各种知识。

    # Java 虚拟机底层原理知识总结

    https://github.com/doocs/jvm

    这里仅仅记录了一些笔者认为需要重点掌握的 JVM 知识点，如果你想更加全面地了解 JVM 底层原理，可以阅读周志明老师《深入理解 Java 虚拟机——JVM 高级特性与最佳实践(第 2/3 版)》全书。

    # 全栈手册

    https://github.com/frank-lam/fullstack-tutorial (opens new window)

    包括java、pythoon、前端、分布式、机器学习等各种知识。

    简介：

    都说好记性不如烂笔头，定期的学习和整理必然对学习巩固有所帮助，这里通过索引的方式对全栈开发技术做一个系统分类，方便随时巩固和学习，当然还有面试。在学习这条路上难免会有很多盲点和学不完的知识。有道无术，术尚可求，掌握好思维能力才能应对千变万化的技术。不要把大脑当成硬盘，也不要做高速运转的 CPU，而修行自己的大脑成为一个搜索引擎，学会分析解决问题。

    # Java工程师进阶知识扫盲

    https://github.com/doocs/advanced-java

    本项目大部分内容来自中华石杉，版权归作者所有，内容涵盖高并发 (opens new window)、分布式 (opens new window)、高可用 (opens new window)、微服务 (opens new window)、海量数据处理 (opens new window)等领域知识。我们对这部分知识做了一个系统的整理，方便学习查阅。

    # CS-Notes

    https://github.com/CyC2018/CS-Notes

    技术面试必备基础知识、Leetcode、计算机操作系统 (opens new window)、计算机网络、系统设计等。

    # JavaGuide

    https://javaguide.cn

    # 小林coding

    https://xiaolincoding.com

    # 代码随想录

    https://programmercarl.com
